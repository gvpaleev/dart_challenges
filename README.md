# Dart Challenges

A collection of Dart programming exercises covering fundamental concepts and practical problem-solving.

## Project Structure

```
dart_challenges/
├── bin/
│   ├── task_1/    # String operations and variable types
│   ├── task_2/    # Loops and conditionals
│   ├── task_3/    # List operations
│   ├── task_4/    # Set operations
│   ├── task_5/    # Map operations
│   ├── task_6/    # Functional programming
│   ├── task_7/    # Object-oriented programming
│   └── task_8/    # Asynchronous programming
└── pubspec.yaml
```

## Requirements

- Dart SDK: ^3.3.0

## Running Tasks

```bash
dart run bin/task_1/main.dart
dart run bin/task_2/main.dart
# ... etc
```

## Task Overview

### Task 1: Strings and Variables
- String interpolation and formatting
- Temperature conversion (Fahrenheit to Celsius)
- Quadratic equation solver
- Variable types: `dynamic`, `var`, `final`, `const`

### Task 2: Control Flow
- FizzBuzz variant ("Super Quiz")
- Count digits without using `length`
- Year to century conversion

### Task 3: Lists
- Sum list elements
- Check list contains value (without `indexOf`)
- List manipulation: sorting, slicing, filtering, spreading

### Task 4: Sets
- Find unique elements and calculate average
- Set operations (symmetric difference)
- Count unique words in text using RegExp

### Task 5: Maps
- Pizza order calculator
- Menu management and sorting
- Using `SplayTreeMap` for sorted maps

### Task 6: Functional Programming
- Sum string lengths using functional methods
- Sum positive numbers only
- Calculate word values based on alphabet positions

### Task 7: Object-Oriented Programming
- Implement `Cuboid` class with volume and surface area
- Implement `Cube` subclass using inheritance
- Getters and constructors

### Task 8: Asynchronous Programming
- Countdown timer with delays
- Using `Future`, `async`, and `await`

## Key Concepts Covered

- **Data Types**: String, int, double, dynamic, var, final, const
- **Collections**: List, Set, Map
- **Control Flow**: if/else, for loops, conditionals
- **Functions**: Regular functions, arrow functions, async functions
- **OOP**: Classes, inheritance, getters, constructors
- **Async**: Future, async/await, delays
- **Functional**: map, reduce, where, forEach
- **String Operations**: Interpolation, formatting, RegExp
